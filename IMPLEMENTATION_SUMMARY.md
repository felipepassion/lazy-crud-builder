## 📋 Usage Pattern

### In LazyCrud (Consumer)

1. **Reference the NuGet packages** containing the templates
2. **Create wrapper .tt files** in your test projects:

```csharp
<#@ include file="..\..\ProjectFiles\Tests\Domain\Tests.Domain.tt" #>
```

3. **Set context** (AggName, namespace, etc.) before including
4. **Generate** using Visual Studio or MSBuild

### Example Wrapper

```csharp
<#@ template debug="false" hostspecific="true" language="C#" #>
<#@ assembly name="System.Core" #>
<#@ import namespace="System.Linq" #>
<#@ output extension=".cs" #>
<#
    // Set aggregate context
    var AggName = "ProductAgg";
    var Namespace = "MyCompany.Product";
#>
<#@ include file="path/to/Tests.Domain.tt" #>
```
